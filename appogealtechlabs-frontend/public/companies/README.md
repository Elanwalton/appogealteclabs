# Company Logos Directory

This folder should contain company logos in SVG format:
- techflow-logo.svg
- bloom-logo.svg
- finedge-logo.svg
- urban-logo.svg
- cargo-logo.svg
- edubright-logo.svg

Recommended specs:
- Format: SVG (preferred) or PNG with transparent background
- Size: 150x40px maximum (will auto-scale)
- Horizontal orientation works best
- Simple, clean designs
